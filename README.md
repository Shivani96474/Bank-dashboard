Project Title:
Digital Payment Transaction Analytics Dashboard

Short Description:
An interactive analytics dashboard designed to monitor and analyze digital payment transactions by month, city, bank, device type, and user demographics, helping identify trends and financial patterns.

Tech Stack:
* Power BI
* DAX
* Microsoft Excel / CSV
* SQL 

Data Source:
Digital transaction dataset containing transaction amounts, balances, bank details, cities, payment methods, device types, age group, and transaction purpose.

Features:
* Monthly transaction trend analysis (Amount & Balance)
* City-wise and currency-wise transaction comparison
* Advanced filters (Bank, City, Device, Gender, Age Group, Payment Method)
* Dynamic line and column chart switching
* Detailed transaction and remaining balance table view
* User-friendly, interactive financial insights for decision-making
